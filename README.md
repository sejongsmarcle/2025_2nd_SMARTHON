# 2025 제2회 SMARTHON
> **2025 제2회 SMARTHON**에 참가하신 여러분을 환영합니다!

## 목표

- 짧은 기간 내 몰입해 특정 문제를 해결하고 개발하는 경험
- 주니어, 시니어를 넘어서 SMARCLE 전 부원 간의 협업 및 네트워킹 경험
- 스터디에서 배운 내용을 실전에 적용하여 부원들의 역량 향상 경험
- 해커톤을 통해 아이디어를 서로 공유하고 협력하며 시야 확장

## 규칙

- **팀 인원 제한**
    - **최대 4명**
    - 모든 팀원 컨택은 슬랙에서만 이루어 지고, 다른 매체에서의 컨택은 인정되지 않습니다.
- **심사 기준 및 방식**
    - **심사 기준**_MVP평가
      ```python
      MVP 완성도 평가 - 총 10점
    
          [10점] 팀에서 목표한 MVP를 얼마나 구현했는가?

      - 계 : 10
      ```
  
    - **심사 기준**_최종발표
      ```python
      아이디어 완성도 - 총 30점
      
          [10점] 아이디어가 실현 가능한가?
          [10점] 추가적인 기능 도입이나 개선을 통한 발전 가능성이 있는가?
          [10점] 아이디어가 얼마나 독창적이고 창의적인가?
              - (현재 시장에서 찾아보기 어려운 독특하고 새로운 아이디어)
    
      구현 완성도 - 총 30점
    
          [10점] 프로젝트의 필수 기능이 구현되어있는가?
          [10점] 예상치 못한 상황과 오류를 얼마나 잘 처리 하였는가?
          [10점] AI 모델과 구현하고자 하는 시스템의 융합이 실질적으로 이루어졌는가?
              - (시스템: 웹/앱, 로봇 등)
    
      주제 적합성 - 총 30점
    
          [10점] 활용한 데이터에 대한 이해가 충분하고 적절한 데이터를 사용했는가?
          [ 5점] 모델 선정 과정에서 명확한 근거와 논리를 제시했는가?
          [ 5점] 모델 선택 과정이 프로젝트 목표와 환경에 맞춰 합리적으로 이루어졌는가?
          [ 5점] 프로젝트 목표에 적합한 성능 지표를 제시하고 이를 명확하게 설명했는가?
          [ 5점] 실제로 학습한 AI모델의 성능이 유의미한가?
    
      발표 전달력 - 총 10점
    
          [ 5점] 발표 속도가 적절하고 적절한 표현으로 발표하였는가?
          [ 5점] 발표자료가 직관적이고 이해하기 쉽게 구성되었는가?
    
      - 계 : 100
      ```

    - 각자 개인물품 미리 준비(예: 칫솔, 치약, 담요 등)
    - 생각보다 대회장 내부가 춥습니다.
